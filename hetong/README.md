### 精装修合同

能看到这，还是上边的话，要么不买，要么签！精装修标准都不会写清楚，只有同档次等模糊词，但是，购房都有一个质量保证书，这个是政府统一的，会写清楚保修的范围及时间；厨卫等电器会由厂家保修，三包政策，是否可以开发商额外提供保修这个可尝试争取。
